# Selenium Camp Examples

### Requirements 
	* Ruby 2.0 or greater
	* Bundler gem
	* Mac OS (need iOS simulator)
	* Appium ($ npm install -g appium)
	
### Install
	* $ git clone git@github.com:isonic1/selenium_camp.git
	* $ bundle install
	
### Run
	* $ appium (start server in seperate terminal)
	* $ ruby proxy-example.rb
